Java List Documenttaion

A comprehenssive guide to theList interface in java.

Introdution

In Java, a List is an interface in the java.util package that represents an ordered collection of elements. It allows duplicate elements and provides index-based access to its elements. Lists maintain the insertion order and are widely used in Java applications.

Key Characteristics

  • Ordered Collection: Elements are stored in a specific
  • Index-Based Access: Retrieve or modify elements using their index.
  • Allows Duplicates:A list can contain multiple occurrences of the same element.
  • Resizable:Dynamically grows or shrinks as elements are added or removed.
  • Iterable:Supports iteration using loops or iterators.

Common Implementations

Implementation Description
ArrayList Backed by a dynamic array; provides fast random access.
LinkedList Implemented as a doubly-linked list; efficient for frequent insertions and deletions.
Vector Thread-safe implementation similar to ArrayList
Stack LIFO (Last-In-First-Out) stack implemented as a subclass of Victor

Common Methods

add(E element)

Adds an element to the end of the list.

get(int index)

Returns the element at the specified index.

remove(int index)

Removes the element at the specified index.

set(int index, E element)

Replaces the element at the specified index with the given e

size()

Returns the number of elements in the list.

Example Code

                 import java.util.ArrayList;
                    import java.util.List;
                    
                    public class ListExample {
                        public static void main(String[] args) {
                            // Create a List using ArrayList
                            List fruits = new ArrayList<>();
                    
                            // Add elements to the list
                            fruits.add("Apple");
                            fruits.add("Banana");
                    
                            // Access elements by index
                            System.out.println("Element at index 0: " + fruits.get(0));
                    
                            // Modify an element
                            fruits.set(1, "Grape");
                    
                            // Print the list
                            System.out.println("Updated list: " + fruits);
                        }
                    }
